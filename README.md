# GorillaLocomotion
[The locomotion system used in Gorilla Tag](https://github.com/Another-Axiom/GorillaLocomotion), only updated for newer UnityEngine versions.

This is made for Unity 6.3 and should work for other Unity 6.X versions.
##
This software is based on the [original project](https://github.com/Another-Axiom/GorillaLocomotion) by Another-Axiom.

Modifications and additional features have been made.
